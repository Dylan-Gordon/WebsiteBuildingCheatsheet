This code snippet turns the very ugly default style of a elements into something much nicer. The navbar you see above is actually just 4 href's in an unordered list. Styling can make a huge difference.


CSS Selector Purpose Example code
. Used to select classes in your html page .classA { ... } -> will select ALL classA's in your html file and adjust their style
table.classA { ... } -> will only select tables that have classA listed as one of their classes.
# Used to select ids in your html page #idA { ... } -> will select ALL idA's in your html file and adjust their style
table.idA { ... } -> will only select tables that have idA listed as one of their classes.
* Used to select all elements * { font-size: 12px; } -> will change the font-size of all elements to be 12 pixels.
:hover Used to specify styles for an element when the item is hovered on. Ex: Hover over elements in this table. table.darkTheme tr:hover {
background-color: #575656;
}
